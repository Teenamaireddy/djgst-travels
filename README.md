# DJGST Travels

Live Website: https://teenamaireddy.github.io/djgst-travels/

DJGST Travels is a ticket reservation website  for bus, train and flight booking.
which is developed by polytechnic students. teena maireddy along with her 
team. their goal is to provide users a user friendly and a single platform to book 
Tickets easily for bus,train and flight.
